# Heart-Disease-Prediction
A machine learning project predicting heart disease using KNN, Decision Trees, and Logistic Regression.

## Overview
This repository showcases my data science and machine learning projects, focusing on applying statistical and computational techniques to solve real-world problems.

## Featured Project: Heart Disease Prediction

### Problem
Predict the likelihood of heart disease based on patient health indicators.

### Approach
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Model training using:
  - K-Nearest Neighbours (KNN)
  - Decision Trees
  - Logistic Regression  
- Model evaluation using AUC, accuracy, and precision  

### Results
Logistic Regression achieved the best performance with the highest AUC (0.924), indicating strong predictive capability.

## Tech Stack
- Python
- Pandas, NumPy  
- Scikit-learn  
- Data visualisation libraries  

## Key Takeaways
- Compared multiple models to identify the most effective approach  
- Built an end-to-end machine learning workflow  
- Strengthened understanding of model evaluation and selection
